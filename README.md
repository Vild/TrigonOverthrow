# TrigonOverthrow
Trigon Overthrow is an angled top down shooter that revolves around the world of 3D primitives.

## Introduction
TurtleGL is a 3D engine written in C++, using OpenGL, for a school project.


## Dependencies
This project uses:
- SDL2
- SDL2_image
- OpenGL
- GLAD
- GLM
- Assimp
